XcodePaste

Power Up Your Paste

A powerful Xcode Source Editor Extension that transforms how you paste text into your code. Save time and write cleaner code with essential text transformations.

Get XcodePaste

Features

📋

Paste as Base64

Convert any text—or image data from your clipboard—into a Base64 encoded string. Perfect for embedding data directly in your code.

📝

Paste as String Literal

Automatically escape quotes, newlines, and special characters. Paste HTML, JSON, or any multi-line text directly as a valid Objective-C or Swift string literal.

🌐

Paste as Unicode

Convert non-ASCII characters (including emojis) to their Unicode escape sequences. Great for internationalized apps.

🔗

Paste as URL Encoded

Transform URLs with Chinese or special characters into properly encoded strings. No more manual URL encoding!

📦

Paste JSON as ObjC

Convert JSON strings directly into Objective-C dictionary, array, or string literals. Paste API responses as ready-to-use code instantly.

⚡

Sort Imports

Automatically organize your import statements. Group framework imports (<>) and local imports (""), then sort alphabetically.

Paste JSON as ObjC Literal

Clipboard Content (JSON):

{
  "users": [
    {"name": "Alice", "id": 1},
    {"name": "Bob", "id": 2}
  ]
}

After Paste (ObjC):

@{
    @"users": @[
        @{@"name": @"Alice", @"id": @1},
        @{@"name": @"Bob", @"id": @2}
    ]
}

#import "Helper.h"

Installation

1

Download and open XcodePaste.app

2

Click "Enable XcodePaste" to open System Settings

3

Ensure "XcodePaste Xcode Source Editor" is checked in Extensions

4

Relaunch Xcode — find XcodePaste in the Editor menu

Download from GitHub App Store

Requirements

  • macOS 11.0 or later
  • Xcode 12 or later

Customize Keyboard Shortcuts

Open Xcode Preferences → Key Bindings → search "xcodepaste" → assign your favorite shortcuts.
